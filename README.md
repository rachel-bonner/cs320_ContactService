# cs320_ContactService
CS 320 Portfolio Reflection – Contact Service
Overview

This project demonstrates my work in software testing, automation, and quality assurance through the development of a Contact Service application. The system allows users to create, update, and delete contact records while enforcing strict validation rules for each field. The included unit tests verify that the application behaves correctly and meets all specified requirements. This project highlights my ability to build reliable software and ensure its correctness through testing.

Ensuring Functional and Secure Software

I ensured my code was functional and secure by writing comprehensive unit tests for both the Contact and ContactService classes. These tests check valid inputs as well as edge cases, such as invalid IDs, overly long fields, and null values. I also implemented validation directly in the classes to prevent incorrect data from being stored. By combining validation and testing, I was able to reduce potential errors and improve the reliability of the system.

Interpreting User Needs

I interpreted user needs by translating the project requirements into clear validation rules and service behaviors. For example, I ensured that contact IDs are unique and cannot be modified, and that fields such as name, phone number, and address follow specific constraints. These decisions help maintain data consistency and usability. Focusing on user needs ensured the system was not only functional but also practical and easy to use.

Approach to Software Design

My approach to designing this software involved breaking the system into smaller, manageable components and applying object-oriented principles. I separated responsibilities between the Contact class and the ContactService class to improve organization and maintainability. I also used unit testing as a core part of development to verify functionality as I built the system. In the future, I would continue using test-driven development and iterative improvements to strengthen both design and code quality.
